# Next.js_14 Project
* live link of the project ( https://next-js-14-flax.vercel.app/ )
  
* Next.js 14
  
First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```


run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
